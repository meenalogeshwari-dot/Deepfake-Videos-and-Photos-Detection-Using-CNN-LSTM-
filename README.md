# Deepfake Detection Project

This project detects whether an image or video is REAL or FAKE using Deep Learning.

## Features
- Image detection
- Video detection
- CNN model
- Flask web application
- Confidence score output

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Flask
- NumPy

## How to Run (Anaconda)

1. Open Anaconda Prompt

2. Activate environment:
conda activate your_env_name

3. Go to project folder:
cd your_project_path

4. Install dependencies:
pip install -r requirements.txt

5. Run the application:
python app.py

6. Open browser:
http://127.0.0.1:5000/

## Output
- Detects REAL or FAKE
- Displays confidence score

## Dataset
Dataset is not included due to size limitations.
It is required only for training the model.

## Note
The trained model (.h5 file) is included, so the application can run without dataset.
